# Tweet Classifier

Description:

Project to build a model which is able to classify the sentiment of tweets.

Code and the output of each code can be viewed in the notebooks

If the code and outputs cannot be viewed directly from Github, please use the following link to load the notebooks:
    
    https://nbviewer.jupyter.org/github/spence998/Tweet-Sentiment-Predictor/tree/master/
    
# Output

The final predictions can be found in the file Outcome.csv in the Additional
files used and output folder.
    
The model predicts the sentiments with an accuracy of 54%.

# Requirements

To run the code copy and paste all files from folder "Files" into the folder where the code is run and change the set directory at the top of every code to this location.

Requirements:

    Packages - Pandas               - Numpy
             - Scikit learn         - MatplotLib
             - NLTK                 - Gensim
             - Tweepy               - Pickle
